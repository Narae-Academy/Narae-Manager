# Narae Manager

직업훈련기관의 과정 운영, 코드 관리, 시간표, 훈련일지, 출석부, 강의계획서 및 PDF 출력을 통합하는 Windows 데스크톱 프로그램입니다.

## 1차 목표

- Python 설치 없이 실행되는 Windows EXE
- 기관·연도·훈련과정·회차 관리
- 과정별 강사·강의실·교과목 코드 관리
- 고용24 업로드용 시간표 자동 생성
- 강사·강의실 충돌 검사
- 과정별 훈련일지 Excel/PDF 생성
- 주말·대한민국 공휴일 자동 제외

## 핵심 데이터 원칙

같은 강사라도 고용24 코드는 과정마다 다를 수 있으므로 다음 조합으로 독립 관리합니다.

`기관 + 연도 + 훈련과정 + 회차 + 강사/강의실/교과목 코드`

## 기술 구성

- .NET 8
- WPF
- SQLite
- ClosedXML
- QuestPDF
- GitHub Actions Windows 자동 빌드

## 개발 단계

1. 프로젝트 골격 및 자동 빌드
2. 과정·회차·코드 데이터베이스
3. 시간표 자동생성 및 충돌검사
4. 훈련일지 Excel/PDF 생성
5. 출석부·강의계획서·평가표·수료증
6. 대시보드 및 AI 업무지원
